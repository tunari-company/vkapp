Политика конфиденциальности

Сервис «АИ Контент — Ведение Пабликов» (мини-приложение ВКонтакте, далее — «Сервис»). Настоящая политика описывает, какие данные обрабатывает Сервис и как они защищены.

1. Какие данные мы обрабатываем

  • Идентификатор пользователя ВКонтакте (vk_user_id) — для создания учётной записи и учёта баланса токенов.
  • Токен доступа ВКонтакте, который вы предоставляете при подключении сообщества (права: стена, фотографии, список сообществ) — исключительно для публикации записей с изображениями от имени выбранных вами сообществ по вашему запросу. Токен необходим потому, что загрузка изображений на стену возможна только с токеном пользователя-администратора.
  • Токен сообщества (по желанию) — для авто-ответов на сообщения сообщества, если вы включаете эту функцию.
  • Созданный вами контент: темы, ссылки на статьи, сгенерированные тексты, изображения и видео, а также расписание публикаций.

Мы используем токен только для публикации в выбранные вами сообщества. Мы не читаем вашу личную переписку и не публикуем ничего без вашей команды.

2. Как хранится токен доступа

Токен доступа хранится в зашифрованном виде (AES, схема Fernet) и используется только для публикации в подключённые вами сообщества. Токен хранится, пока сообщество подключено, и удаляется при его отключении. Вы можете отозвать доступ приложения в любой момент в настройках ВКонтакте: Настройки → Настройки приложений, после чего Сервис утратит возможность публикации.

3. Кому передаются данные

Для генерации контента текст ваших запросов (тема поста, содержимое статьи, описание изображения или видео) передаётся сторонним провайдерам ИИ исключительно для создания текста, изображений и видео по вашему запросу:

  • OpenAI — генерация текста и изображений;
  • Anthropic (Claude) — генерация текста;
  • Google (Gemini) — генерация изображений;
  • Seedance / ByteDance — генерация видео.

Провайдеры обрабатывают запросы только для формирования результата. Данные не используются для обучения моделей, не передаются третьим лицам в маркетинговых целях и не продаются.

4. Удаление данных

Отключение сообщества в интерфейсе Сервиса удаляет сохранённый токен доступа из наших систем. Чтобы полностью отозвать доступ приложения на стороне ВКонтакте, откройте Настройки ВКонтакте → Настройки приложений. Для полного удаления учётной записи и всех данных напишите в сообщество поддержки Сервиса — данные будут удалены в течение 30 дней.

5. Изменения политики

Актуальная версия политики всегда доступна по этому адресу. Дата обновления: 12 июля 2026 г.
